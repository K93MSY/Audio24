# Audio24
### An tool always Send random sound file.  

### build  
```
git clone https://github.com/K93MSY/Audio24/
npm install express
node index.js
```

### How to use
 put Mp3file in ./sounds/an/
 rename ___filenumber___.mp3
 add detail in sound-info.json
